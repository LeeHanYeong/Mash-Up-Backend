# Mash-Up Backend

[![codecov](https://codecov.io/gh/LeeHanYeong/Mash-Up-Backend/branch/master/graph/badge.svg)](https://codecov.io/gh/LeeHanYeong/Mash-Up-Backend)

### - [API문서](https://mashup.lhy.kr/doc/)



## Requirements

- Python >= 3.7
- Poetry >= 1.0



## Installation

```
poetry install
```



## Test

```
pytest app
```



## Coverage (codecov.io)

> 환경변수로 `CODECOV_TOKEN`이 필요합니다

```
pytest --cov app
codecov
```
